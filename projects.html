My Projects

A collection of projects that I love to do in my free time...

#1 Expense Tracker

The project integrates a frontend built with React, managing global state using Context API, and a backend built with Node.js and Express.js, using Axios for HTTP communication and MongoDB for data storage. This architecture provides a scalable and maintainable solution for tracking expenses with a user-friendly interface.

Built on:

  • React.js - for Frontend
  • Express.js - for middleware
  • Node.js - for Backend
  • MongoDB - for Database
  • GitHub

#2 Student Record

The Student Record System combines frontend development using Microsoft Expression Web 4 with backend functionality powered by a MySQL database. The system is deployed on a playground server using FTP for easy access and management. It provides a user-friendly interface for managing student records efficiently.

Built on:

  • Microsoft Expression Web 4 - for Frontend
  • MySQL - Database
  • FTP - to access on global network
  • Uploaded on playground server
  • GitHub

#3 Post Secondary Grad Rates

This project was a part of Data Visualizations project which examines data and trends at 3,800 degree-granting institutions in the United States (excluding territories) that reported a first-time, full-time degree-seeking undergraduate cohort, had a total of at least 100 students at the undergraduate level in 2013, and awarded undergraduate degrees between 2002 and 2013.

Included:

  • Outliers
  • Data Kurtosis
  • Graphs
    1. Pie Chart
    2. Seaborn Lineplot
    3. Scatter Plot
    4. Seaborn Bar Plot
  • GitHub
